def derivada(ec,variable,h=0.01):
    df = (ec(variable + h) - ec(variable))/h
    return df

El siguiente código encuentra las raíces usando el método de Newton y la ecuación programada en el inciso 2. Hace el método hasta que el punto evaluado sea menor que el error.

In [83]:
def newton(ec,p,err=0.001):
    
    raiz = p
    while abs(ec(raiz)) > err:
        raiz = raiz - ec(raiz)/derivada(ec,raiz)
    
    return raiz

def newton_bessel(ec,p,err=0.001):
    
    raiz = p
    while abs(ec(raiz)) > err:
        raiz = raiz - ec(raiz)/ec4(raiz)
    
    return raiz

5.-¶

Sea una matriz

$$ \left( \begin{array}{ccc} a & b \\ c & d \end{array} \right) $$

que veremos aquí como:

[[a,b],[c,d]]

Sabemos que si el determinante es cero la matriz no tiene inversa, para lo cual se usa el primer if de la función. Si no, voltearemos los de la diagonal y les cambiaremos el signo a los otros.

In [162]:
def inversa(matriz):
    
    if matriz[0][0]*matriz[1][1] - matriz[0][1]*matriz[1][0] == 0:
        print "No tiene matriz inversa"
        
    else:
        det_i = 1/(matriz[0][0]*matriz[1][1] - matriz[0][1]*matriz[1][0])
        a = matriz[1][1]*det_i
        b = -matriz[0][1]*det_i
        c = -matriz[1][0]*det_i
        d = matriz[0][0]*det_i
        
        print [[a,b],[c,d]]

Vemos del siguiente ejemplo que efectivamente nos da la matriz inversa. Es importante escribir los números con un punto porque si no no hace bien la división del determinante.

Queremos primero convertir nuestro número en un entero, para eso si no es divisible entre uno lo multiplicamos por diez hasta que lo sea.

Luego le restamos uno a nuestro número hasta que sea divisible entre diez y cada paso que se hace lo vamos sumando a una "n". Cuando es divisible entre diez lo dividimos entre diez y repetimos. Así se suma cada dígito.

In [314]:
def sumador(num):
    n = 0
    while num%1 != 0:
        num = num*10
        
    while num >= 1:
        while num%10 != 0:
            num = num - 1
            n += 1
        num = num*0.1
    
    return n
